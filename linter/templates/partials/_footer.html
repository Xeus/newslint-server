Newslint is a project by Ben Turner, created with Python and Django, under a dev environment using Bower, Grunt, Github, and SASS, hosted on Amazon EC2 with Apache, Varnish, and memcached, as a fun side project to try integrating different frameworks and tools. Newslint is inspired by Rowan Manning's Javascript version of joblint, for which I also made a Python port.
