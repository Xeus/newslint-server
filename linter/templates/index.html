What is Newslint?

In consumption and analysis of the news, half the battle is deciphering the intent and agenda behind the text to see if it's giving the story a fair take. Newslint identifies loaded language, potential biases, punditry, partisanship, unprofessionalism, and other problems with text. Not all of its warnings will apply, but hopefully it will make you aware of issues with the text.

You can make pull requests to improve the rules for detection, via the git repo and you can also use the site's API.
