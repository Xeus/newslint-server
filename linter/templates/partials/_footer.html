Newslint is a project by Ben Turner, created with Python, Django, Bower, and SASS, hosted on Amazon Elastic Beanstalk, as a fun side project to try integrating different frameworks and tools. It is inspired by Rowan Manning's Javascript version of joblint, for which I also made a Python port.
